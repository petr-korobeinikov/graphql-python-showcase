# graphql-python-showcase

### Stop using REST, Post-JSON, and other stuff from 2k00.

## Running

```shell
docker build -t graphql-python-showcase .
docker run --rm -p 9000:9000 graphql-python-showcase
```

Open `http://<your-docker-host>:9000/`.
